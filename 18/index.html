テーマ１８

ゲームを作りたいのですが、時間がなくて完成させることができません。ということで、こんな感じでやっています。パスファインダーアルゴリズムを利用したアニメーションのようなものです。 このゲームの基本は、パスファインダーアルゴリズム（A*アルゴリズム）をどのように実装するかということです。このプログラムでは、そのアルゴリズムを使って、ある地点に移動するキャラクターを見ることができます。

スクリーンショット

ユーザー向けの説明

キャラクタは、1フレームごとに使用する画像を変えてアニメーションさせる。

  • 最初は、プログラムが自動的にキャラクタに目的地を指定する。
  • 目的地に着くと、椅子に座る。
  • また歩き出すために、画面をクリックするだけである。


開発者向けの説明

表示されている画像を変化させることでアニメーションを作成します。
使用する画像を変更するために、ファイル名を格納するStringフィールド String fileName; を使用した。ロードされた画像ファイルがこのフィールド値に基づいていることを意味する。
このフィールドは、他のフィールド、String name; String action; String direction; int index; によって決定される。
this.fileName = "res\\" +this.name + "_" + this.action + "_" + this.direction + "_" + this.index + ".png";
このプログラムは、繰り返し実行するたびに、これらのフィールドの値を変化させる。
例えば、キャラクターが右に移動する場合、this.setDirection("right"); 方向フィールドは "right"に変更されます
キャラクターが目的地に到着すれば、this.setAction("sit"); フィールドの値は "sit "に変更されます。
