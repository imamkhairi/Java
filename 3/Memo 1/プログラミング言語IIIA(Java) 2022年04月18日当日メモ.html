課題1: ファイル名 XYVector.java
class XYVector {
    // ----まずフィールド(変数)を宣言----
    double x, y;
    // ----その下にメソッド(関数)を作る----
    void print() {
    ~~~~これを忘れずに!!
        System.out.println(x);
        System.out.println(y); もっと凝った表示にしてもOK
    }
}

課題2: ファイル名 XYVectorApp.java
class XYVectorApp {
    public static void main(String[] args) {
        XYVector a;
        a = new XYVector();　メモリを割り当ててインスタンス化する
        a.x = Double.parseDouble( args[0] );
        a.y = Double.parseDouble( args[1] );
        a.print(); これで print メソッドを呼び出す
    }
}
$ javac XYVector.java XYVectorApp.java
$ java XYVectorApp 3.14 2.17 引数が二つ必要
